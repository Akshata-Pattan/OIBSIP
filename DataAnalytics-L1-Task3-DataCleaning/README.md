# Data Cleaning and Preprocessing

## Objective
This project focuses on cleaning and preprocessing a real-world retail dataset by handling missing values, removing duplicate records, converting data types, and standardizing text columns.

## Tools Used
- Python
- Pandas
- NumPy
- Google Colab

## Dataset
Real-World Online Retail Dataset

## Data Cleaning Steps
- Checked dataset information and summary statistics.
- Identified missing values.
- Filled missing categorical values using mode.
- Filled missing numerical values using median.
- Removed duplicate records.
- Converted OrderDate to datetime format.
- Standardized text columns by removing extra spaces and applying title case.
- Saved the cleaned dataset.

## Files Included
- Data_Cleaning_and_Preprocessing.ipynb
- online_retail_real_world.csv
- Cleaned_Online_Retail_Dataset.csv

## Output
A cleaned and preprocessed dataset ready for Exploratory Data Analysis (EDA) and Machine Learning.

## Conclusion
The dataset was successfully cleaned and preprocessed, improving its quality and making it suitable for further analysis.
