# Exploratory Data Analysis (EDA) on Retail Sales Dataset

## Objective
Perform Exploratory Data Analysis (EDA) on a retail sales dataset to discover trends, customer behavior, and business insights using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset
Retail Sales Dataset

## Key Analysis
- Data Cleaning
- Missing Value Check
- Statistical Summary
- Monthly Sales Trend
- Quarterly Sales Trend
- Customer Age Distribution
- Gender Distribution
- Revenue by Product Category
- Correlation Heatmap
- Box Plot Analysis

## Conclusion
The analysis provides valuable insights into sales trends, customer demographics, and product performance, helping businesses make data-driven decisions.
